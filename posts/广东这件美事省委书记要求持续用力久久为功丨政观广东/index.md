# 广东这件“美”事，省委书记要求持续用力、久久为功丨政观广东


> 南方杂志微信公众号 [原文链接](https://mp.weixin.qq.com/s/IVqo6QuBE1kjAtTKYMRr-w)

　　党的十八大以来，习近平总书记对广东<font color="#ff0000">生态文明建设</font>念兹在兹，历次视察广东都对抓好<font color="#ff0000">生态文明建设</font>作出重要指示要求，指出广东有条件有能力把<font color="#ff0000">生态文明建</font>设搞得更好。

　　<span style="background:#fff88f">广东认真贯彻落实习近平总书记、党中央决策部署，把<font color="#ff0000">生态环境保护</font>摆在全局工作突出位置来抓，<font color="#ff0000">系统谋划、全面部署、强力推进</font>，确保始终沿着总书记指引的方向推进<font color="#ff0000">生态文明建设</font>。</span>

　　站在新起点上，广东不仅要绿，还要美。

　　怎么做好这件“美”事？<span style="background:#fff88f">2023年9月22日，全省生态环境保护大会暨<font color="#ff0000">绿美广东生态建设</font>工作会议在广州召开，部署以<font color="#ff0000">绿美广东生态建设</font>为牵引，<font color="#ff0000">全面推进广东生态文明建设</font>，奋力<font color="#ff0000">打造人与自然和谐共生的<strong>中国式现代化</strong>广东样板</font>。</span>

　　省委书记黄坤明指出，要牢牢抓住<font color="#ff0000">绿美广东生态建设</font>这一战略举措，持续用力、久久为功。

　　<font color="#ff0000">环境就是民生，青山就是美丽，蓝天也是幸福。</font>广东将“绿美”绣进发展蓝图，这是广东生态文明建设的破题“新解法”，正在开启<font color="#ff0000">新时代生态文明建设</font>的新篇章。

------

### 绿美广东为何如此重要

　　背枕南岭、面朝南海，珠江奔流、水网密布，降水丰沛、热力充足，植物繁茂、四季花开……在一代又一代人的接续努力下，广东森林覆盖率达到53.03%，森林面积较40年前增长67%。绿色，日益成为广东高质量发展的鲜明底色和澎湃动能。

　　迈上新征程，广东经济社会发展已进入<span style="background:#fff88f">加快<font color="#ff0000">绿色化、低碳化</font>的<font color="#ff0000">高质量发展</font>阶段</span>，但生态文明建设仍处于压力叠加、负重前行的关键期。如何提升发展“含绿量”、“含金量”？亟待以更高站位、更宽视野、更大力度来谋划和推进<font color="#ff0000">生态文明建设</font>。

　　<span style="background:#fff88f"><font color="#ff0000">绿美广东</font>，就是这张新答卷的“题眼”。</span>

　　去年，广东省委十三届二次全会作出深入推进绿美广东生态建设的重要决定。今年，<span style="background:#fff88f"><font color="#ff0000">绿美广东生态建设</font>进一步被纳入<font color="#ff0000">省委“1310”具体部署</font></span>，省委书记带头推进。全省各地以<font color="#ff0000">绿美广东</font>为牵引，<font color="#ff0000">生态文明建设</font>加快推进。

　　<span style="background:#fff88f">“<strong><font color="#ff0000">绿美广东生态建设</font>是<font color="#ff0000">广东生态文明建设</font>的战略牵引，也是<font color="#ff0000">关系广东长远发展和民生福祉的重要工程</font>。</strong>”</span>在全省生态环境保护大会暨<font color="#ff0000">绿美广东生态建设</font>工作会议上，黄坤明进一步道出<font color="#ff0000">绿美广东生态文明建设</font>在全省发展中的重要性。

> 2025年12月7日，广东省考申论二大作文题目——<span style="background:#fff88f">习近平总书记指出：“良好生态环境是最普惠的民生福祉。”</span> 请结合给定材料，联系实际，围绕<span style="background:#fff88f">“<strong>绿美广东生态建设</strong>是关系广东<strong>长远发展</strong>和<strong>民生福祉</strong>的重要工程”</span>进行深入思考，自拟题目，撰写一篇议论文。
>
> 正如上文所述，<span style="background:#fff88f"><font color="#ff0000">绿美广东</font>，就是这张新答卷的“题眼”。</span>

　　重要在何处？可以从三个方面来认识。

　　<span style="background:#fff88f">绿美广东是<font color="#ff0000">习近平生态文明思想</font>在广东的落地生根，<font color="#ff0000">饱含着总书记对广东的殷切厚望</font>。</span>要从讲政治的高度来加强认识，把坚持“两个确立”、做到“两个维护”体现到加强生态环境保护、推进广东生态文明建设的实际行动上，切实肩负起使命责任，知重负重、知难克难、知责尽责，守护好广东的绿水青山，不辜负总书记的厚望重托。

　　<span style="background:#fff88f">绿美广东是推动<font color="#ff0000">广东在中国式现代化建设中走在前列</font>的应有之义。</span>我们所追求的<font color="#ff0000">中国式现代化</font>，是<font color="#ff0000">人与自然和谐共生的现代化</font>。<strong>广东要奋力<font color="#ff0000">打造人与自然和谐共生的中国式现代化广东样板</font>，就要把<font color="#ff0000">建设生态文明</font>当作必须答好的<font color="#ff0000">必答题</font>。</strong>而绿美广东是一步“先手棋”，是广东生态文明建设的总纲、总抓手、主动力、主方向。

　　<span style="background:#fff88f">绿美广东是<font color="#ff0000">广东扎扎实实搞生态、谋发展、添后劲的必由之路</font>。</span>事实证明，<font color="#ff0000">生态本身就是经济，保护生态就是发展生产力</font>。当前，广东正在举全省之力推动<font color="#ff0000">高质量发展</font>，解决生态环境突出问题是必须跨越的重要关口。<span style="background:#fff88f">只有推动绿美广东生态建设向纵深发展，才能不断塑造发展的新动能、新优势，持续增强发展的潜力和后劲。</span>

------

### 如何做好这件“美”事

　　从“绿”到“美”，每个人都能切身感受到生态向好的分量。

　　根据《中共广东省委关于深入推进绿美广东生态建设的决定》，到2027年底，广东将完成林分优化提升1000万亩、森林抚育提升1000万亩，全域建成国家森林城市，率先建成国家公园、国家植物园“双园”之省。

　　到2035年，全省完成林分优化提升1500万亩、森林抚育提升3000万亩，多树种、多层次、多色彩的森林植被成为南粤秀美山川的鲜明底色，天蓝、地绿、水清、景美的生态画卷成为广东亮丽名片，绿美生态成为普惠的民生福祉。

　　蓝图已经绘就，如何久久为功、绵绵用力？黄坤明指出五个方向。

　　一是<span style="background:#fff88f">要加力提速、攻坚突破，推动<font color="#ff0000">绿美广东生态建设</font>向纵深发展，用心用情绘就岭南山川城乡绿美实景图。</span>

　　二是<span style="background:#fff88f">要坚持重点突破与系统推进相结合，大力推进<font color="#ff0000">绿美广东生态建设</font>“六大行动”，突出林分优化、林相改善提质量，突出点、线、面结合优布局，突出管、防、控并重强防护，持续推动绿美广东生态建设走深走实、见行见效。</span>

　　三<span style="background:#fff88f">要坚持<font color="#ff0000">生态美、产业强、文化兴、百姓富</font>相促进，持续推进点绿成金、深化造绿为民、加强以绿养心，不断提升<font color="#ff0000">绿美广东生态建设</font>综合效益。</span>

　　四<span style="background:#fff88f">要坚持改革创新与守住红线相统一，稳步推进集体林权制度改革，持续深化国有林场改革，加快深化造林、护林、用林机制改革，努力为<font color="#ff0000">绿美广东生态建设</font>增动力、添活力。</span>

　　五是<span style="background:#fff88f">要坚持强氛围、扩主体、拓途径相贯通，通过强化内涵挖掘、打造品牌活动、推进互学互促浓厚全民绿化美化的参与氛围，充分发挥党政机关、企事业单位、团员青年和志愿者以及各民主党派、工商联、无党派人士的积极作用，不断扩大全民绿化美化参与主体和参与途径，切实提高绿美广东生态建设全民参与度。</span>

　　方向已明，目标已定，关键就看行动。

　　广东是大家的广东，只要我们一起干好<font color="#ff0000">绿美广东生态建设</font>这件“美”事，一定能在广袤大地上种下片片绿色，也在广大人民心中播撒绿色种子，让南粤大地天更蓝、山更绿、水更清、生态更优美，推动美丽中国的广东样板日渐成形。
