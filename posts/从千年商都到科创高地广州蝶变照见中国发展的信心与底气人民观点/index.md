# 从“千年商都”到“科创高地”


> 本文节选自《人民日报》2025年12月10日
>
> 作者：本报评论部

　　城市是现代化建设的重要载体，也是观察中国式现代化的重要窗口。关于广州的两则新闻发人深思——

　　今年前10个月，广州进出口总额破万亿元，创历史同期新高。

　　不久前，《自然》增刊公布全球十强科研城市排名，广州跃升至全球第六，超越美国旧金山湾区。

　　<span style="background:#fff88f">从<font color="#ff0000">“千年商都”</font>到<font color="#ff0000">“科创高地”</font>，广州的蝶变正是中国<font color="#ff0000">高质量发展</font>的生动缩影。</span>

　　<span style="background:#fff88f">习近平总书记指出：“要紧紧抓住<font color="#ff0000">高质量发展</font>这个首要任务，全面推进<font color="#ff0000">中国式现代化</font>。”</span>中国大地上，690多座城市禀赋各异、产业多元，立足全国大局中的战略定位，各展其长，优势互补，走出一条中国特色城市现代化新路子，必将汇聚成推进强国建设、民族复兴伟业的强大势能。

　　“中国发展前景是光明的，我们有这个底气和信心。”

------

### （一）

　　<span style="background:#fff88f">信心，在<strong>以<font color="#ff0000">科技创新</font>引领发展<font color="#ff0000">新质生产力</font></strong>，为<strong>以<font color="#ff0000">高质量发展</font>全面推进<font color="#ff0000">中国式现代化</font></strong>注入不竭动能。</span>

　　前不久<span style="background:#fff88f">在广东考察时，习近平总书记强调：“着眼<font color="#ff0000">发展新质生产力</font>，<strong>强化<font color="#ff0000">科技创新</font>和<font color="#ff0000">产业创新</font>深度融合</strong>，建设具有国际竞争力的<font color="#ff0000">现代化产业体系</font>。”</span>

　　<span style="background:#fff88f"><font color="#ff0000">科技创新</font>是发展<font color="#ff0000">新质生产力</font>的核心要素。</span>推动科技创新这个“关键变量”转化为<font color="#ff0000">高质量发展</font>的“最大增量”，创新城市优势明显、活力满满。

　　<span style="background:#fff88f"><font color="#ff0000">十五运会“源火”</font>来自<font color="#ff0000">深海可燃冰</font>，这是一场<font color="#ff0000">产学研用的协同攻关</font>，也是相关产业的合力托举</span>：广州海洋地质调查局提供深海作业手段及平台保障，企业开发高效光伏追光系统，<span style="background:#fff88f"><font color="#ff0000">首艘大洋钻探船“梦想”号</font>完成<font color="#ff0000">“引天之光，燃海之冰”</font>的壮举。</span>

　　围绕<font color="#ff0000">产业链</font>部署<font color="#ff0000">创新链</font>，围绕<font color="#ff0000">创新链</font>布局<font color="#ff0000">产业链</font>。从南沙科学城到中新知识城，一条轴线串联起广州30%的高校科研院所、51%的高新技术企业、74%的独角兽企业，引领老城广州走向全新未来。

　　<span style="background:#fff88f">重大创新周期长、风险大、难度高，<strong>推动<font color="#ff0000">有效市场</font>与<font color="#ff0000">有为政府</font>更好结合</strong>，形成支持全面创新的基础制度，创新生态更有生命力。</span>

　　小鹏汽车董事长何小鹏分享过一个创新接力的故事。首次创业，举步维艰，广州天河区科技工业和信息化局雪中送炭，让企业熬过生死寒冬。二次创业，何小鹏毫不犹豫再选广州，并打造出跻身新能源汽车第一方阵的小鹏汽车。

　　今年前10个月，<span style="background:#fff88f">广州的<font color="#ff0000">低空经济</font>与<font color="#ff0000">航空航天</font>、<font color="#ff0000">未来网络</font>与<font color="#ff0000">量子科技</font>、<font color="#ff0000">深海深空</font>、<font color="#ff0000">智能无人系统</font>等领域，新登记经营主体增速均超100%。<font color="#ff0000">新兴产业</font>加速发展，<font color="#ff0000">未来产业</font>快速崛起，正是<font color="#ff0000">创新土壤肥沃</font>与<font color="#ff0000">产业舞台广阔</font>的见证。</span>

　　放眼全国，“新”风扑面。安徽合肥加速量子科技产业化发展，湖北武汉加速打造世界级光电子信息产业集群，宁夏中卫在戈壁滩打造“智算绿洲”……科技创新与产业创新融合互促，新质生产力加速发展。

　　<span style="background:#fff88f"><strong>以<font color="#ff0000">创新</font>之“脉”，壮<font color="#ff0000">产业</font>之“骨”</strong></span>，这正是我们赢得未来、赢得长远的关键所在。

------

### （二）

　　<span style="background:#fff88f">信心，在<strong>以<font color="#ff0000">高水平开放</font>助力<font color="#ff0000">新发展格局</font></strong>加快形成，牢牢把握发展主动权。</span>

　　<span style="background:#fff88f">坚持开放合作、互利共赢是<font color="#ff0000">中国式现代化</font>的必然要求。习近平总书记强调：“要不断<font color="#ff0000">扩大高水平对外开放</font>，深度参与全球产业分工和合作，用好国内国际两种资源，拓展<font color="#ff0000">中国式现代化</font>的发展空间。”</span>

　　当前，单边主义、保护主义抬头，国际经济贸易秩序遇到严峻挑战。越是外部环境不确定性增多，越要加快构建新发展格局。

　　<span style="background:#fff88f"><strong>以<font color="#ff0000">制度创新</font>打造开放新高地，增添经济新活力</strong>。</span>

　　广州南沙打造“离岸易”综合服务平台，建立“白名单”制度并出台专项政策对重点企业提供结算便利。自挂牌起，广东自贸试验区南沙片区累计形成45项在全国复制推广的制度创新成果，充分彰显了改革“试验田”作用。

　　“十四五”时期，新疆外贸进出口总值连续跨过3个千亿元台阶，打造亚欧黄金通道和向西开放桥头堡；广西以高水平共建西部陆海新通道牵引开放发展，2024年通道沿线省份经广西进出口贸易额比2019年增长了近一倍。实行更加积极主动的开放战略，高水平开放促进高质量发展。

　　<span style="background:#fff88f"><strong>以<font color="#ff0000">供应链</font>携手推动<font color="#ff0000">合作共赢</font>，拓展发展新空间</strong>。</span>

　　广汽以智能座舱设计拓展西欧市场，万宝冷柜用蓄电池供电应对非洲的停电现象……今年前10个月，广州对欧盟、东盟的进出口分别增长23.9%、31.1%；对共建“一带一路”国家和金砖国家进出口增长均超20%。市场多元化成效明显，激发“买全球、卖全球”的新活力。

　　放眼神州大地，长江黄金水道打开江西出海通道，中欧班列让陕西从内陆腹地变开放前沿……全国统一大市场打破“壁垒”、突破“阻隔”，各个城市不再受限于区位，都能在扩大开放中增优势、添活力，确保中国经济“稳”的基调不变，“进”的势头不减，“韧”的底色不改。

　　<span style="background:#fff88f"><font color="#ff0000">畅通国内国际双循环</font>，提高在全球配置资源能力，更好争取开放发展中的战略主动，这是我们穿越风雨、稳步前行的坚实底气。</span>

------

### （三）

　　<span style="background:#fff88f">信心，在提升<font color="#ff0000">高质量发展</font>动力源作用，形成更多重点区域增长极。</span>

　　我国经济发展的空间结构正在发生深刻变化，中心城市和城市群正在成为承载发展要素的主要空间形式。

　　习近平总书记强调，“要形成几个能够带动全国<font color="#ff0000">高质量发展</font>的新动力源”。2024年中央经济工作会议提出，发挥<font color="#ff0000">区域协调发展</font>战略、区域重大战略、主体功能区战略的叠加效应，积极培育新的增长极。大城市、城市群要勇担重任，抢抓政策红利，打造发展新引擎。

　　<span style="background:#fff88f"><font color="#ff0000">胸怀大局</font>才能<font color="#ff0000">挑起大梁</font>。</span>

　　<span style="background:#fff88f">排头兵、领头羊、火车头，3个关键词寄托着习近平总书记对广州的殷切期望。</span>今年，从<font color="#ff0000">首款复合翼飞行汽车</font>完成首飞，到<font color="#ff0000">飞行汽车智造基地</font>竣工，<font color="#ff0000">低空经济</font>加速“起飞”。从“智车之城”迈向“天空之城”，广州找准大局中的定位乘势而上。

　　“十四五”时期，两条母亲河、三大城市群，既有联通南北东西的经济轴带，也有引领<font color="#ff0000">高质量发展</font>的重要动力源，与四大区域板块融合协同，全国一盘棋，<font color="#ff0000">高质量发展</font>动力澎湃。

　　<span style="background:#fff88f"><font color="#ff0000">错位发展</font>才能各展所长。</span>

　　<span style="background:#fff88f">今年<strong><font color="#ff0000">“深圳—香港—广州”创新集群</font>创新指数跃居全球第一</strong>。<font color="#ff0000">三地“硬联通”“软联通”“心联通”</font>，不靠单打独斗靠系统集成，不靠资源堆砌靠生态协同。现代化都市圈在大湾区崛起，兄弟齐心、其利断金。</span>

　　<span style="background:#fff88f">推进<font color="#ff0000">中国式现代化</font>，必须用好<font color="#ff0000">区域协调发展</font>的辩证法，以先发地区的“一马当先”引领全国各地发展的“万马奔腾”。</span>

　　今天的中国，<font color="#ff0000">京津冀</font>把发展落差的势能变成协同发展的动能，<font color="#ff0000">长三角</font>推动<font color="#ff0000">科技创新</font>和<font color="#ff0000">产业创新跨区域</font>协同，<font color="#ff0000">成渝地区双城经济圈</font>为推进新时代西部大开发提供支撑……覆盖东中西，协调南北方，一个个增长极不断涌现，形成“1+1>N”的整体效应，这正是我们实现<font color="#ff0000">高质量发展</font>的独特优势。

　　<span style="background:#fff88f">明天得到什么，取决于昨天有什么，更取决于今天干什么。</span>

　　<span style="background:#fff88f">面向<font color="#ff0000">“十五五”</font>，以<font color="#ff0000">科技创新</font>强<font color="#ff0000">动力</font>，以<font color="#ff0000">新发展格局</font>激<font color="#ff0000">活力</font>，以<font color="#ff0000">区域协调发展</font>聚<font color="#ff0000">合力</font>，推进<font color="#ff0000">中国式现代化</font>建设，战胜风险挑战、实现宏伟蓝图，我们信心十足、底气十足、力量十足。</span>
