# 江门向海图强 海工装备产业强势崛起丨走读“百千万工程”③


> 南方杂志微信公众号 [原文链接](https://mp.weixin.qq.com/s/F_hBWNqlKRpe9oCLwdR4uA)
>
> 《南方》杂志全媒体记者｜左志红

<span style="background:#fff88f"><font color="#ff0000">“百县千镇万村高质量发展工程”</font>（以下简称<font color="#ff0000">“百千万工程”</font>）是<font color="#00aaff">广东破解城乡区域发展不平衡难题</font>、推动<font color="#ff0000">高质量发展</font>的关键所在。<strong>今年是<font color="#ff0000">“百千万工程”</font>三年初见成效之年</strong>，各地即将交上怎样的答卷？《南方》杂志记者深入全省多地，探访“百千万工程”落地见效举措，探寻广东城乡区域协调发展的生动实践。</span>

------

　　站在江门新会崖门水道岸边，记者眼前<span style="background:#fff88f">这个高如9层楼、长达155米的“钢铁巨兽”，正是<font color="#ff0000"><strong>全球首艘自航式水体自然交换型深远海养殖工船“湾区伶仃”号</strong></font>，年产与陆地5000亩鱼塘相当，这艘<font color="#ff0000">“养殖航母”</font>5月29日在中交四航局江门航通船厂顺利下水。</span>不远处，南洋船舶的船坞内工人们正赶制出口的节能型小灵便散货船，该公司在这个领域市场占有率居全球第一，订单已排至2028年……

　　一艘艘<font color="#ff0000">海工重器</font>挺进“深蓝”、一座座<font color="#ff0000">海洋牧场</font>拔“海”而起、一家家<font color="#ff0000">造船企业</font>昼夜繁忙……这是江门深入实施“百县千镇万村高质量发展工程”三年来的生动场景。<span style="background:#fff88f">江门是全省海洋大市、重要造船基地，在<font color="#ff0000">珠三角沿海城市</font>中<font color="#ff0000">大陆岸线最长</font>、<font color="#ff0000">海岛数量最多</font>、<font color="#ff0000">滩涂面积最大</font>。</span>近年来，江门发挥海洋大市优势，努力打造“海上新江门”，绘出一份“海工装备领航、产业聚链成群、深江协同破局”的“蓝色”画卷！

------

### 海工装备破圈：大国重器频出 产业集聚创新力全球领先

　　江门，临江面海、水网交织，这座因水而兴的城市，造船历史源远流长。早在南宋，新会就能制造“载重700至1000石”的横江船，我国历史上著名三大船型之一的“广船”主要在江门新会区生产。1954年江门造船厂建厂后，江门一跃成为广东省内仅次于广州的第二大造船基地。

　　<span style="background:#fff88f"><font color="#ff0000">船舶与海工装备制造</font>，是江门重点发展的<font color="#ff0000">战略性新兴产业</font>。近年来，江门抓住<font color="#ff0000">海洋强国、海洋强省建设战略</font>机遇，进一步<font color="#ff0000">发挥海洋大市资源禀赋</font>，加快推动<font color="#ff0000">产业高端化、集群化发展</font>，成为<font color="#ff0000">全省船舶企业数量最多</font>的城市之一。</span>

　　<span style="background:#fff88f"><font color="#ff0000">“大国重器”</font>启航。全球最大宽扁浅吃水型半潜驳船、国内首套潜水无人切割机、华南最大全回转起重船等高端装备在江门诞生</span>；2024年造船完工量99艘，较2023年翻番，产业链增速达59.7%，连续3年保持两位数高速增长。

　　<span style="background:#fff88f"><font color="#ff0000">“江门制造”</font>出海。</span>今年一季度船舶出口额同比增长54.6%；1—4月，广东最大民营造船企业——南洋船舶同比增长近40%，在细分领域市场占有率全球领先；海星游艇连续7年进入全球80英尺以上超级游艇榜单前30强。

　　<span style="background:#fff88f"><font color="#ff0000">“创新引擎”</font>驱动。江门航通<font color="#ff0000">首创“电力推进+自然海水交换”养殖模式</font>，<font color="#ff0000">“湾区伶仃”号实现远程投喂、AI监测和“海上游牧”</font>；<font color="#ff0000">华南最大螺旋桨制造厂——开平远航</font>，是全球为数不多生产8.9米可调螺旋桨的厂家。</span>

　　<span style="background:#fff88f">串珠成链、集群成势，江门船舶与海工装备“硬核”破圈。</span>去年12月，“中国海洋第一展”——中国海博会在深圳举行。江门作为省内唯一受邀地级市亮相，20多家江企展出多项世界领先水平产品，赢得各方瞩目。

------

### 深江协同发展：强强联合 形成“深圳总部+江门基地”合作模式

　　<span style="background:#fff88f"><font color="#ff0000">粤港澳大湾区</font>是全国三大船舶产业聚集区之一，深圳被赋予“建设全球海洋中心城市”的重大使命。随着<font color="#ff0000">深中通道</font>开通，深圳江门两座海洋城市之间的合作按下“加速键”，在去年底正式签署海洋经济发展合作框架协议。</span>

　　优势互补、强强联合。<span style="background:#fff88f">江门坐拥4880平方公里的蔚蓝海域、415公里的金色海岸，扼守崖门、虎跳门出海口和西江、潭江“黄金水道”，海洋资源得天独厚，是<font color="#ff0000">全省最具海洋经济发展优势的城市</font>之一。</span>

　　<span style="background:#fff88f">深圳、江门海洋经济优势互补、各有所长，两市正在积极构建<font color="#ff0000">“深圳总部+江门基地”、“深圳研发+江门生产”、“深圳服务+江门制造”合作模式</font>，推动知名的船舶与海工装备制造企业总部落地深圳、生产基地落户江门。</span>

　　协同破局、携手前行。<span style="background:#fff88f">围绕打造深圳全球海洋中心城市江门基地，两市加快规划建设66平方公里的<font color="#ff0000">深圳·江门经济合作区</font>，共建<font color="#ff0000">大湾区船舶与海工装备产业基地</font>，推动大广海湾经济区打造成为国家新一轮开放合作平台。</span>

　　在广海湾基地，江门利用3.2公里岸线和6.5平方公里港区，与深圳加快发展高端船舶及海工装备制造、特种船舶维修、船舶配套关键零部件等产业，联合对接国内国际先进资源，打造高端船舶及海工装备综合基地。

　　由深圳海斯比公司在台山生产基地完成建造的<font color="#ff0000">中国首个移动浮岛“圆周号”</font>，正是深圳江门海洋产业合作的生动案例。

------

### 海洋牧场进军：六位一体 加快打造全省现代化海洋牧场先行示范市

　　在台山川岛乌猪洲高冠海域，江门制造的全省首座入级中国船级社的大型现代化养殖装备——“台山1号”如钢铁城堡般矗立，在3万立方米水体中，投放的数万尾良种鮸鱼翻涌如银浪，耕海牧渔、乘风破浪！

　　以海工装备为关键支撑，江门加快打造全省现代化海洋牧场先行示范市，全产业链推进深远海养殖，累计建成现代化深水网箱389个，去年海洋牧场产量同比增长22.68%、排名珠三角第一，海洋牧场规模化集群化发展。

　　<span style="background:#fff88f"><font color="#ff0000">“蓝色种业”</font>攻关。</span>全市新增渔业种业企业30多家，携手中山大学建设海洋发展优势种业与智慧养殖联合研发中心，培育推广适合江门海域养殖的鮸鱼、章红鱼、石斑鱼等高价值品种，首批鱼获成功上市、成活率100%。

　　<span style="background:#fff88f">水产加工“卖全球”。</span>江门水产品加工业增加值21.86亿元，拥有全国最大的鳗鱼养殖生产和出口基地，形成饲料生产、鳗鱼养殖、烤鳗加工和鳗鱼出口“一条龙”产业链，鳗鱼、生蚝等预制菜“游”向全球。

　　<span style="background:#fff88f"><font color="#ff0000">“渔港经济”</font>崛起。</span>广东五大渔港之一的广海渔港推进渔港升级和美丽渔村建设，渔港综合管理试点、生态海堤等项目相继完成，投资1.5亿元建设的广海水产交易中心试运营，加快打造国家级现代渔港经济区。

　　<span style="background:#fff88f"><font color="#ff0000">“风渔”</font>融合发展。</span>江门抢抓川岛海上风电项目建设契机，探索培育绿色氢、氨、醇“制造—存储—运输—应用”新能源产业，推进<font color="#ff0000">“海上风电+海洋牧场”</font>融合开发。

　　<span style="background:#fff88f"><font color="#ff0000">“种苗培育—深海养殖—精深加工—装备制造—融租服务—海上风电”六位一体</font>，江门<font color="#ff0000">深耕第一产业、壮大第二产业、拓展第三产业，“三线并进”</font>加快构建<font color="#ff0000">现代海洋产业体系</font>，走出具有江门特色的<font color="#ff0000">海洋经济高质量发展</font>之路。</span>

------

### 一线观察：江门从“海洋资源大市”到“海洋强市”

　　从崖门水道到广海湾，江门的“蓝色崛起”绝非偶然。这里有<font color="#ff0000">“向海图强”</font>的战略定力，还有<font color="#ff0000">“海洋生态保护”</font>的全局眼光，更有<font color="#ff0000">“敢闯新路”</font>的实干基因。

　　<span style="background:#fff88f"><strong><font color="#ff0000">向海图强</font>、挺进深蓝</strong></span>！江门做好经略海洋文章、加快海洋强市建设，正在“打造海上新江门”上实现新突破。
